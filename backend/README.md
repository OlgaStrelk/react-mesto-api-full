# Проект Mesto бэкенд
RESTful API для проекта "Mesto" - интерактивной страницы, где пользователи могут делиться фотографиями.

## Реализованы

* подключение к базе данных 
* роутинг
* авторизация

## Использованные технологии
* Express.js
* nodemon
* MongoDB
* mongoose
* dotenv
* cors
* celebrate, Joi
* validator
* bcryptjs
* jsonwebtoken
* eslint

## Можно улучшить
